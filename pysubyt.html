class pysubyt.api.ReIterableAccess(*args, **kwargs)[source]#

Bases: dict

Helper class wraps around an existing dict of iterables, making it posible to access the members so that they return independent object wrappers that insure totally independent iterating loops into them

update([E, ]**F) → None.  Update D from dict/iterable E and F.[source]#

If E is present and has a .keys() method, then does: for k in E: D[k] = E[k] If E is present and lacks a .keys() method, then does: for k, v in E: D[k] = v In either case, this is followed by: for k in F: D[k] = F[k]
